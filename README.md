# Sito Web del Dott. Stefano Forlani

Questo sito web è stato creato per fornire informazioni sul Dott. Stefano Forlani e i suoi servizi medici.

## Tecnologie utilizzate

* HTML
* CSS
* Bootstrap
* Firebase Hosting

## Funzionalità

* Pagina iniziale con informazioni di contatto e presentazione del medico.
* Sezione dedicata ai servizi offerti.
* Sezione con informazioni sulle specializzazioni.
* Modulo di contatto per prenotazioni e richieste di informazioni.
